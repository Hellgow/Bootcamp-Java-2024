        <h2>Descrição do Projeto</h2>
        <p>Este é um projeto de simulação de um processo seletivo em Java, onde são realizadas tentativas de contato com candidatos e uma seleção baseada em salário pretendido.</p>
        
        <h2>Funcionalidades</h2>
        <ul>
            <li>Realiza tentativas de contato com os candidatos;</li>
            <li>Analisa o salário pretendido dos candidatos;</li>
            <li>Seleciona os candidatos adequados para a vaga;</li>
            <li>Imprime os candidatos selecionados.</li>
        </ul>
